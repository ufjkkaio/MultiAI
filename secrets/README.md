# APIキーを置く場所

このフォルダは、**APIキーなど秘密情報を置く場所**です。

## 使い方

1. `.env.example` をコピーして、このフォルダ内に **`.env`** という名前で保存する。
2. `.env` を開き、各キーを記入する。
   - `OPENAI_API_KEY=sk-...`（OpenAI / ChatGPT）
   - `GEMINI_API_KEY=...`（Google Gemini）
3. **`.env` は絶対に git にコミットしない。**（.gitignore で除外済み）

---

## 安全のために（必ず守ること）

- **`.env` を GitHub や他の人に送らない。** 漏れるとキーが悪用されます。
- **git でコミット・プッシュする前に** `git status` を実行し、`.env` が一覧に**出てこないこと**を確認する。出ていたら `git reset` で外す。
- キーはコードに直書きせず、バックエンドでは環境変数（この `.env` を読み込む）からだけ参照する。
- 本番サーバーでは、ホスティング先の「環境変数」にキーを登録し、サーバー上の `.env` は使わない運用も検討する。
