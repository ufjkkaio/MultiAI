プライバシーポリシー

[サービス名]（以下「本サービス」）は、ユーザーの個人情報の保護を重視しています。

1. 収集する情報

本サービスでは、以下の情報を収集・利用することがあります。

  • 認証情報：Sign in with Apple により、Apple から提供されるユーザー識別子および、ユーザーが選択した場合の表示名・メールアドレス。
  • 会話内容：本アプリ内でやり取りしたメッセージ（ユーザーの送信内容およびAIによる返答）。
  • 利用状況：利用回数などの統計的な情報。

2. 利用目的

収集した情報は、本サービスの提供、お問い合わせへの対応、サービスの改善・障害対応、法令に基づく対応のみに利用します。第三者に販売することはありません。

3. 保存期間

会話内容は、アカウント削除または法令等に基づく要請があるまで保存します。アカウント削除時に削除します。

4. 第三者提供

個人を特定できる情報を、法令に基づく場合を除き、第三者に提供することはありません。本サービスの提供にあたり、OpenAI および Google（Gemini）の API に会話内容を送信します。各社のプライバシーポリシーに従って取り扱われます。
