# MultiAI サービス：理念と単価シミュレーション

## 1. 理念の整理

### 製品
- **複数のAIとグループチャットができるサービス**
- 1つのルームに複数AIを参加させ、ユーザーと一緒に会話する

### 解決したいこと
- 「AIは必ずしも正しくない」と知っていても、
  - 複数AIの登録・入力・前提説明・それぞれのサブスク
  - が邪魔して、結局1つのAIに留まってしまう
- → **シンプルな料金で、複数AIとのグループチャット**を届け、その不満を埋める

### 作る理由
- 自分に便利だと思った
- ネットに詳しい自分でもそんなサービスを知らなかった
- ニッチで終わらせず、「凡人でも知ってる・使える」サービスにしたい
- 自分が知らなかった＝まだ余地がある、という判断

### 届く価値（ユーザーにとって）
- **雑談を複数AIとできる**：1つのルームで、気軽に複数AIと会話し続けられる。
- **情報の精度が上がる**：いろんなAIがいるから、視点や答えが複数得られ、確認・補完しやすい。
- **過信を避けられる**：1体だけだと「これが正解」と信じがち。複数いれば「AIも必ずしも正しくない」が自然に伝わる。

---

## 2. 料金イメージ

- **サブスク：1980〜3000円/月**（中央値 約2500円で試算）
- **グループ＝異なるAIが最低2体以上**（例：Gemini + ChatGPT）。1プロバイダだけではグループにならない。
- 各AIは **高速・低コスト** のモデルを想定（例：Gemini Flash、GPT-4o mini など）。詳細・制限は今後詰める。

### モデルはプランで固定（ユーザーは選ばない）

- **ユーザーにモデル選択はさせない**。どのプランならどのモデルかは**サービス側で固定**する。
- メリット：単価が読みやすい／UIがシンプル／「どのAIが来るか」で迷わない。
- **より良いモデル**（例：GPT-4o、Gemini 3 Flash）は、**あとから上位プラン**（例：2,980円）を設けて提供する形がよい。

## 3. 1980〜3000円で利益が出るか：概算

**前提**：グループ＝**異なるAIが最低2体**。ここでは **2体（Gemini + もう1社）** で試算する。

### 3.1 前提

- **為替**: 1 USD = 150 JPY（保守的に）
- **1ユーザーあたりの利用**: 「1メッセージ → 2体のAIが返信」（グループの最小構成）
- **トークン想定**: 1メッセージあたり 入力2,000トークン×2回、出力500トークン×2回

### 3.2 パターンA：Gemini Flash + Claude Haiku（2体）

| プロバイダ | 入力単価 | 出力単価 | 1メッセージあたりコスト（USD） |
|------------|----------|----------|--------------------------------|
| Gemini 2.5 Flash | $0.30/100万 | $2.50/100万 | 0.0006+0.00125 = **$0.00185** |
| Claude Haiku 3.5 | $0.80/100万 | $4.00/100万 | 0.0016+0.002 = **$0.0036** |
| **合計（2体/1メッセージ）** | - | - | **$0.00545** |

→ **約 0.82 円/メッセージ**（150円/USD）

### 3.3 パターンB：Gemini Flash + GPT-4o mini（2体・より安い）

| プロバイダ | 入力単価 | 出力単価 | 1メッセージあたりコスト（USD） |
|------------|----------|----------|--------------------------------|
| Gemini 2.5 Flash | $0.30/100万 | $2.50/100万 | **$0.00185** |
| GPT-4o mini | $0.15/100万 | $0.60/100万 | 0.0003+0.0003 = **$0.0006** |
| **合計（2体/1メッセージ）** | - | - | **$0.00245** |

→ **約 0.37 円/メッセージ**

### 3.4 月額イメージ（2体・パターンA：Gemini+Claude）

| 利用量（メッセージ/月） | APIコスト（円/月） | 売上（2,500円） | Apple 15%控除後 | 粗利（API除く） |
|-------------------------|--------------------|------------------|------------------|-----------------|
| 100 | 約 82円 | 2,500円 | 2,125円 | 約 2,043円 |
| 300 | 約 246円 | 2,500円 | 2,125円 | 約 1,879円 |
| 500 | 約 410円 | 2,500円 | 2,125円 | 約 1,715円 |
| 1,000 | 約 820円 | 2,500円 | 2,125円 | 約 1,305円 |

- **Apple 15%**（Small Business Program）を控除した取り分で試算。
- 1980円プランの場合：取り分 約1,683円。月500メッセージでもAPI 約410円 → **約1,273円のマージン**。

### 3.5 その他のコスト（目安）

| 項目 | 月額目安 |
|------|----------|
| 決済 | Apple経由のため別途Stripe等なし（iOSのみ前提） |
| サーバー・DB（小規模） | 総額 1,000〜3,000円/月 程度 |
| ドメイン・その他 | 数百円/年 程度 |

### 3.6 1980円で利益が出る「コスパ最優先」のモデル

| プロバイダ | おすすめモデル | 入力/100万 | 出力/100万 | 備考 |
|------------|----------------|------------|------------|------|
| **OpenAI** | **gpt-4o-mini** | $0.15 | $0.60 | 最安クラス。会話品質も十分。 |
| **Gemini** | **gemini-2.0-flash** | $0.10 | $0.40 | 2.5 Flash より安い。会話向き。 |

**この2つ**（GPT-4o mini + Gemini 2.0 Flash）なら、1メッセージあたりのAPIコストは **約 0.15円**（2体で）。  
月500メッセージでも **約75円**。1980円プラン（Apple 15%控除後 約1,683円）なら **約1,600円以上のマージン**が残る。

- もう少し性能を上げたい場合：Gemini は **gemini-2.5-flash**（入力$0.30・出力$2.50）に変更可能。その分コストは増えるが、1980円でも月500メッセージ程度なら利益は出る。

### 3.7 Gemini 3 Flash を使う場合

**使用可否**: Google の API で提供されていれば **利用可能**。モデル名は `gemini-3-flash` や `gemini-3.0-flash` 等、公式ドキュメントで確認する。

**料金（Gemini 3 Flash）**
- 入力: $0.50 / 100万トークン
- 出力: $3.00 / 100万トークン

**組み合わせ**: GPT-4o mini（据え置き）+ **Gemini 3 Flash**

| 項目 | 1メッセージあたり |
|------|-------------------|
| GPT-4o mini | 2,000×$0.15/1e6 + 500×$0.60/1e6 = **$0.0006** |
| Gemini 3 Flash | 2,000×$0.50/1e6 + 500×$3.00/1e6 = **$0.0025** |
| **合計（2体）** | **$0.0031** → **約 0.47円/メッセージ**（150円/USD） |

**月額イメージ（1980円プラン・Apple 15%控除後 取り分 約1,683円）**

| 利用量（メッセージ/月） | APIコスト（円/月） | 粗利（取り分−API） |
|-------------------------|--------------------|----------------------|
| 100 | 約 47円 | 約 1,636円 |
| 300 | 約 141円 | 約 1,542円 |
| 500 | 約 235円 | 約 1,448円 |
| 1,000 | 約 470円 | 約 1,213円 |

**結論**: Gemini 3 Flash を使っても **1980円で利益は十分発生する**。2.0 Flash よりコストは増えるが、月500メッセージでも約1,450円のマージンが残る。

**採用モデル（固定）**: **GPT-4o mini + Gemini 3 Flash** で運用する場合、上記の試算のとおり利益は見込める。上限 2,000〜3,000 でも全員使い切っても黒字。

### 3.8 価格のマーケット心理（1980 / 2000 / 2300 / 2500円）

| 価格 | 心理効果の目安 |
|------|----------------|
| **1980円** | **「2000円以下」の心理的ハードルを下回る。** チーム価格としてよく使われる。「お得感」が伝わりやすく、**購買意欲を引き出しやすい**。 |
| 2000円 | キリがいいが、「2000円」はひとつの心理的ライン。1980より「高い」と感じられがち。 |
| 2300円 | 端数価格。やや高めに感じる層が増える。 |
| 2500円 | さらに高め。「本気で使う人」向けにはあり。 |

**結論（購買意欲）**: **1980円**がもっとも「買いやすい」と感じられやすい。次点で 2000円。2300・2500は単価は上がるが、成約率は落ちる可能性が高い。まずは **1980円** で試し、反応を見てから上位プランで 2500円などを設けるのが現実的。

### 3.9 結論（採算）

- **GPT-4o mini + Gemini 3 Flash の固定でも、1980円で利益は見込める。**
- **2体（異なるプロバイダ）でも、1980〜3000円のサブスクで収益は発生する。**
- **コスパ最優先**なら **gpt-4o-mini + gemini-2.0-flash**。1980円で十分マージンが残る。
- **Gemini 3 Flash** を使っても1980円で利益は出る（月500メッセージで約1,450円のマージン）。性能を重視する場合は選択肢になる。
- 利用上限（月あたりメッセージ数など）を設ければ、ヘビーユーザーでも採算が崩れにくい。
