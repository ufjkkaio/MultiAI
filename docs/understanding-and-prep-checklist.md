# 方向性・サービス内容の整理と、あなたが準備すること

## 1. こちらが理解していること

### サービスの中身
- **複数AIとのグループチャット**：1つのルームに自分＋複数AIが参加し、同じスレッドで雑談・相談し続ける。
- **価値**：雑談できる／複数視点で情報の精度を上げられる／1体への過信を避けられる。シンプルな料金で「複数AI」のハードルを下げる。
- **差別化**：競合は「1問→複数回答を比較」。こちらは「グループチャットでずっと会話する」体験。

### ビジネスまわり
- **料金**：月額 1,980〜3,000円のサブスク（想定）。
- **プラットフォーム**：iOSアプリのみ（まずはここで）。
- **課金**：App Store のサブスク。**Small Business Program 参加前提**（手数料15%）。
- **会員**：サブスクなので会員（認証）は必須。

### 技術・運用の前提
- **「グループ」＝異なるAI（異なるプロバイダ/モデル）が最低2体以上**。Geminiだけではグループにならない。価値（複数視点・過信を避ける）も、**別々のAI**がいてこそ成り立つ。
- そのため **最低でも2プロバイダ**（例：Gemini + Claude、または Gemini + OpenAI）が必要。各社のAPIキー・利用規約の準備が前提。
- 各AIのモデルは「高速・低コスト」を優先（例：Gemini Flash、Claude の軽量系など）。詳細・制限はあとで詰める。
- デザインは凝らさず、まず **動くもの** を優先。

## 2. あなたが準備するとよいこと

### すぐあると助かるもの
| 項目 | 内容 |
|------|------|
| **Apple Developer アカウント** | 有料（年間約12,800円）。アプリ公開・IAP・Small Business Program 申請に必要。 |
| **複数プロバイダのAPIキー** | **グループにするため最低2つ**。例：Gemini（Google AI Studio / Cloud）+ Claude（Anthropic）または OpenAI。各社の利用規約・課金設定を確認。 |
| **Small Business Program** | App Store Connect から申請。年間収益100万ドル以下が条件。 |

### 作る前に決めたいこと（後からでも可）
| 項目 | 例 |
|------|-----|
| **アプリ名・サービス名** | 例：MultiAI など。 |
| **サブスクの価格** | 1,980円 / 2,980円 など、1本か複数プランか。 |
| **参加させるAIの組み合わせ** | 例：Gemini + Claude。または Gemini + GPT。最低2体・異なるプロバイダが必須。 |

### 作るときに都度でよいもの
| 項目 | 内容 |
|------|------|
| **デザイン・文言** | 画面の雰囲気、ボタンや説明文の希望。なければシンプルで進める。 |
| **利用制限** | 月あたりメッセージ数など。MVP ではゆるくしてあとで詰めても可。 |

## 3. まとめ

- **方向性・サービス内容は理解している**：複数AIとのグループチャット、サブスク・iOS・App Store・Small Business Program 前提。
- **あなたの準備**：**Apple Developer** と **異なるプロバイダのAPIキーを最低2つ**（例：Gemini + Claude または Gemini + OpenAI）が揃えば、実装を始められる。アプリ名・価格は早めに決めるとスムーズだが、仮で進めてあとで変えてもよい。
